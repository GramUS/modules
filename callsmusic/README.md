# Calls Music

Stream music in group calls.

---

## Installation
```text
.update gramjsuserbot/plugins/callsmusic
```

## Notes
- Requires FFmpeg.
- Commands should be sent in groups.

## Commands
|Command|Description|
|------|-|
|stream|Stream the replied audio file/voice message.|
|stop|Leave the call.|
|resume|Resume the stream.|
|pause|Pause the stream.|
